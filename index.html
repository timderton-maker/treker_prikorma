Важно: данные хранятся только в браузере

На сервер ничего не передаётся. Но браузер может их удалить:

  • 📱 iOS Safari — удаляет через 7 дней без посещения сайта
  • 📱 iOS Chrome / Яндекс — аналогично (используют движок Safari)
  • 🤖 Android — данные могут удалиться при очистке кеша
  • 🖥️ Любой браузер — «Очистить данные сайтов» удалит всё

📲 Совет: добавь сайт на главный экран — это защитит данные от удаления на iOS и откроет его как приложение!

🍎 iPhone / iPad Safari → кнопка «Поделиться» → «На экран Домой»
🤖 Android Chrome / Яндекс → три точки → «Добавить на главный экран»
🔍

Как работает поиск?

  • 🥦 По названию — начни вводить слово, и сайт найдёт нужный продукт
  • 🔢 По возрасту — введи число (например 6), и покажет все продукты, которые можно с этого возраста
  • 👆 Нажми на результат — сайт автоматически прокрутит к карточке и раскроет категорию
